漢文返り点トレーニング

数字カードを文の横に置いて、読む順番をつかむ。

1. まず大前提：漢文は「読む順番」を先に決める

漢文でつまずきやすい原因は、いきなり意味を考えようとすることです。先にやることは、現代語訳ではありません。

基本の流れ
① 返り点を見る → ② 読む順番を数字で決める → ③ 書き下し文を確認する → ④ 意味を考える
  • 返り点は、日本語の語順で読むための「戻る印」です。
  • 置き字は、声に出して読まない字です。ただし、意味やつながりを考えるヒントになります。
  • 書き下し文では、「を・に・より・て」などの助詞や送り仮名を補います。

2. 返り点の種類と考え方

返り点は、返る範囲がだんだん大きくなります。今回扱うのは甲乙点までです。

返り点返る大きさ読み方の基本イメージ
レ点 すぐ近く すぐ下の字を読んでから、レ点のついた字へ戻る。 不レ 知 は「知 → 不」
一二点少し離れる一点を読んでから、二点に戻る。「一 → 二」の順に戻る。
上下点大きいまとまり一二点より大きなまとまりで、上点から下点の順に読む。「上 → 下」の順に読む。
甲乙点さらに大きいまとまり上下点よりさらに大きなまとまりで、甲点から乙点の順に読む。「甲 → 乙」の順に読む。

※要点例の 不レ知 では、縦書きで見てもレ点は「不」の左下（横）につきます。「知」につくのではありません。

上下点・甲乙点の注意：上下点は「上 → 下」、甲乙点は「甲 → 乙」の順に読みます。ただし、これは単に上から読むという意味ではなく、一二点だけでは足りない大きな返り方を整理するための記号です。

3. 読む順番の決め方

  • レ点は、となり同士の入れ替え。
  • 一二点は、少し離れた語を入れ替える。
  • 上下点は、一二点より外側の大きなまとまり。
  • 甲乙点は、上下点よりさらに外側のまとまり。
ここだけ覚える
小さい範囲から順に処理します。
レ点 → 一二点 → 上下点（上→下） → 甲乙点（甲→乙）

4. 置き字の基本

置き字は、漢文の中に書かれているけれど、書き下し文では基本的に読まない字です。

ただし、「読まない＝無視していい」ではありません。置き字は、場所・比較・接続・文末の意味を考えるための目印になります。

置き字の考え方
① 声に出して読まない。
② でも、意味を考えるヒントにはする。
③ 書き下し文では「に・より・て・して」などを補うことがある。

よく出る置き字一覧

置き字主な働き書き下し文での出方例
於場所・対象・比較・起点を表す目印。字そのものは読まない。文脈により「に」「より」「を」などを補う。青出二於藍一
→ 青は藍より出づ
于於と似て、場所や対象を表す。基本は読まない。「に」「より」などを考えるヒント。行于道
→ 道に行く
乎場所・対象・疑問や反語の語気を表すことがある。置き字としては読まないことが多い。文末では疑問の感じを添える。学不亦説乎
→ 学びて時に之を習ふ、亦説ばしからずや
而前後をつなぐ。順接「そして」、逆接「しかし」のような働き。読まないが、「て」「して」「しかして」などでつながりを補う。学而不レ思
→ 学びて思はず
焉文末で断定・強調・詠嘆の感じを添える。場所の意味を含むこともある。基本的に読まない。文末のニュアンスを考えるヒント。終焉
→ 終はる、など文脈で判断
矣文末で完了・断定・強調を表す。基本的に読まない。「〜た」「〜である」のような意味を考えるヒント。学成矣
→ 学成れり

5. 再読文字の基本

再読文字は、一つの漢字を二回読む特別な字です。今回の教材では、まず代表だけ確認します。

漢字読み方意味例
未いまだ〜ずまだ〜ない未レ知 → 未だ知らず
将まさに〜んとす今にも〜しようとする将レ行 → 将に行かんとす
当まさに〜べし当然〜すべきだ当レ学 → 当に学ぶべし
まとめ：返り点は読む順番の印、置き字は読まないけれど意味のヒントです。まずは「どこに返るか」を見ることから始めましょう。
